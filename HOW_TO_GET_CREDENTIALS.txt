================================================================================
TESTING YOUR CREDENTIALS
================================================================================

After creating both .env files:

1. Open 2 terminals
2. Terminal 1:  cd web && npm run dev
3. Terminal 2:  cd api && python -m uvicorn main:app --reload

4. Visit: http://localhost:3000
5. Try signing up - if it works, credentials are correct!

If you see Firebase errors:
  - Check .env values match your Firebase project
  - Make sure all required fields are present
  - Verify PRIVATE_KEY has proper newlines
