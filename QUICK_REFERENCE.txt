GETTING STARTED
================================================================================
1. Run setup:       setup.bat (Windows) or ./setup.sh (Mac/Linux)
2. Add .env files:  See QUICKSTART.md
3. Start servers:   run-dev.bat or ./run-dev.sh
4. Visit:           http://localhost:3000

QUICK ERROR FIXES
================================================================================
Error: "Cannot find module '@tailwindcss/forms'"
Fix:   clean-build.bat (Windows) or ./clean-build.sh (Mac/Linux)

Error: "AxiosError: Network Error"
Fix:   Make sure BOTH servers are running:
       Terminal 1: cd web && npm run dev
       Terminal 2: cd api && python -m uvicorn main:app --reload

CACHE CLEANUP
================================================================================
Clear Next.js cache:       rm -rf web/.next
Clear npm cache:           npm cache clean --force
Clear Python cache:        rm -rf api/__pycache__
Full cleanup script:       clean-build.bat or ./clean-build.sh

PORT CONFLICTS
================================================================================
Port 3000 in use?  Ctrl+C to stop, or use: npm run dev -- -p 3001
Port 8000 in use?  Ctrl+C to stop, or use: python -m uvicorn main:app --reload --port 8001
